## Required

Node 14.x.x^

To install Node use [NVM](https://github.com/nvm-sh/nvm#installing-and-updating):

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

Install Node lts:

```bash
nvm i --lts
```

## Install dependencies

```bash
npm i
```

## Running

```bash
npm start
```

## Building

```bash
docker build -t k8s-js-test-app .
```
